# Discos Ionic Express Sequelize

Proyecto de gestión de discos usando **Ionic (frontend)** + **Node.js / Express (backend)** + **Sequelize** para conexión con base de datos.

Este repositorio contiene tanto el backend (Express + Sequelize) como el frontend (Ionic).

## Getting Started

Estas instrucciones permiten obtener una copia del proyecto funcionando en local para desarrollo y pruebas.

### Prerequisitos

Necesitas tener instalado:

- MySQL
- Visual Studio Code (VS Code)

### Instalación

#### Backend

```bash
cd backend
npm install
npm start
